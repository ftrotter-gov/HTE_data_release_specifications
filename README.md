Health Tech Ecosystem Data Release Specifications
===================

This repository contains data specifications in markdown formats for doing data releases that can be leveraged by the National Provider Directory as part of the healthcare tech ecosystem at CMS.

While this repo is hosted by CMS, it the industry collaboration that has designed and established these optional specifications.

Data release specifications:
- [CEHRT Vendor Reporting](data_release_specifications/CEHRT_Vendor_Reporting.md)
- [General Provider Endpoint and Affiliation Data](data_release_specifications/GeneralProviderEndpointAndAffiliationData.md)
- [How Did The Connection Go](data_release_specifications/HowDidTheConnectionGo.md)
- [Payer Provider Data Without Endpoints](data_release_specifications/Payer_ProviderDataWithoutEndpoints.md)
- [Payer Endpoints](data_release_specifications/PayerEndpoints.md)

If you are releasing data under these specifications with the intention that they should be included in the new [National Provider Directory](https://directory.cms.gov/), please reach out to us through the [CMS Health Technology Ecosystem](https://www.cms.gov/priorities/health-technology-ecosystem/overview) slack or through npd@cms.hhs.gov
