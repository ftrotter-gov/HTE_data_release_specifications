| **Provider Directory Publishing & Usage Metrics**<br><br>**Publish directory of providers (NPI Type 2 organizations, and NPI Type 1 individual providers where available) and other network participants (e.g., networks, payers) through the network, along with their usage stats by provider for each use case, to the National Provider Directory (NPD).**<br><br>In order to create a transparent ecosystem, networks must share which providers participate, their interoperability endpoints, and metrics on their volume and response rate for each usage type (patient, treatment, payment, operations). This information will populate the NPD, enabling:<br>• Ecosystem stakeholders to know where and how to query for information.<br>• Networks to publish participants on the network<br>• CMS to monitor adoption and performance.<br><br>NPD integration requires stable endpoints, NPI-level metadata; if NPI-level metadata is unavailable please provide organizational level metadata, and update cadences established by CMS. Networks should expect to ingest and push updates routinely as more providers join and provider information changes, and should also flag information they find that is incorrect. Specifications for the NPD can be found pinned in the #provider-directory Slack channel. | 6–7 |
